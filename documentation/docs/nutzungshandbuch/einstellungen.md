# Einstellungen

Menü → **Einstellungen**

## Design (Hell-/Dunkelmodus)

Klicke das **Sonnen-/Mond-Symbol** in der AppBar, um zwischen Hell- und Dunkelmodus zu wechseln.

## Passwort ändern

1. Klicke **„Passwort ändern"**.
2. Aktuelles Passwort eingeben.
3. Neues Passwort eingeben und bestätigen (gleiche Regeln wie bei Registrierung).
4. **Speichern**.

## Konto löschen

1. Klicke **„Konto löschen"**.
2. Bestätige den Dialog.
3. Alle Daten werden gelöscht: Meldungen, Fotos, Kommentare, Favoriten, gespeicherte Suchen.

!!! warning "Nicht rückgängig machbar"
    Die Kontolöschung ist endgültig. Alle Daten werden sofort und unwiderruflich entfernt.

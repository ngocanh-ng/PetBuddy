# Nutzungshandbuch

Dieses Handbuch richtet sich an Endanwender und beschreibt alle Funktionen von PetBuddy – von der Registrierung bis zur Verwaltung eigener Meldungen.

---

## Funktionsübersicht

| Funktion | Beschreibung |
|----------|-------------|
| [Registrierung & Anmeldung](registrierung_anmeldung.md) | Konto erstellen, anmelden, abmelden |
| [Entdecken & Suchen](entdecken_suchen.md) | Meldungen durchsuchen, filtern und auf der Karte anzeigen |
| [Meldung erstellen](meldung_erstellen.md) | Vermisste oder gefundene Tiere melden, mit optionaler KI-Rassenerkennung |
| [Kommentare](kommentare.md) | Kommentare lesen, schreiben, beantworten und mit Reaktionen versehen |
| [Meine Meldungen](meine_meldungen.md) | Eigene Meldungen bearbeiten, als wiedervereint markieren, löschen oder als PDF exportieren |
| [Favoriten & Gespeicherte Suchen](favoriten_gespeicherte_suchen.md) | Meldungen merken und Suchfilter für späteren Zugriff speichern |
| [Profil bearbeiten](profil_bearbeiten.md) | Benutzername und Profilbild verwalten |
| [Einstellungen](einstellungen.md) | Design anpassen, Passwort ändern oder Konto löschen |
| [Beispieldaten](beispieldaten.md) | Test-Account und Demonstrationsszenarien zum Ausprobieren |

---

## Gastmodus

PetBuddy kann auch ohne Anmeldung genutzt werden. Im Gastmodus können Sie:

- Meldungen durchsuchen und filtern
- Die Kartenansicht nutzen
- Meldungsdetails und Kommentare lesen

Für alle weiteren Funktionen (Meldungen erstellen, kommentieren, Favoriten setzen etc.) ist eine Anmeldung erforderlich. Im Gastmodus erscheint ein Banner mit einem **Anmelden**-Button oben auf der Startseite.

## Beispiel-Meldungen

In der Datenbank befinden sich bereits Meldungen mit unterschiedlichen Tierarten, Standorten und Status (Vermisst / Gefunden), um die Such- und Filterfunktionen testen zu können.

## Demonstrationsszenarien

| Szenario | Schritte |
|----------|----------|
| **Meldung erstellen** | Anmelden → Melden → Formular ausfüllen → Speichern |
| **Tier suchen** | Start → Filter auf „Vermisst" + Tierart „Katze" + Ort „Berlin" → Ergebnisse ansehen |
| **Favorit setzen** | Meldung in der Liste → Herz klicken → Menü → Favoritenlisten prüfen |
| **Kommentar schreiben** | Meldung öffnen → Kommentar eingeben → Absenden → Antwort testen |
| **KI-Erkennung nutzen** | Melden → Foto hochladen → KI starten → Ergebnis übernehmen |
| **Suche speichern** | Filter setzen → „Suche speichern" → Name vergeben → Menü → Gespeicherte Suche → Anwenden |
| **PDF exportieren** | Meine Meldungen → Meldung wählen → „Exportieren als PDF" |

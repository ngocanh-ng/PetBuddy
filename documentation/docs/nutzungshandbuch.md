# Nutzungshandbuch

<a id="nutzungshandbuch"></a>

Dieses Handbuch richtet sich an Endanwender und beschreibt alle Funktionen von PetBuddy – von der Registrierung bis zur Verwaltung eigener Meldungen.

---

## Registrierung & Anmeldung

### Registrieren

1. Öffne das Seitenmenü (☰) und wähle **Anmelden**.
2. Wechsle zum Tab **Registrieren**.
3. Fülle folgende Felder aus:
    - **E-Mail-Adresse**
    - **Benutzername** (wird öffentlich angezeigt)
    - **Passwort** (mind. 8 Zeichen, Groß-/Kleinbuchstabe, Zahl, Sonderzeichen)
    - **Passwort bestätigen**
4. Klicke auf **Registrieren**.

Nach erfolgreicher Registrierung bist du automatisch angemeldet.

### Anmelden

1. Menü → **Anmelden**
2. E-Mail und Passwort eingeben → **Anmelden**

### Passwort vergessen

1. Im Login-Formular auf **„Passwort vergessen?"** klicken.
2. E-Mail-Adresse eingeben.
3. Du erhältst eine E-Mail mit einem Zurücksetzungslink.

### Abmelden

Über das **Logout-Symbol** (⎋) in der AppBar oder im Menü unter **Abmelden**.

## Entdecken & Suchen

### Übersicht (Start)

Nach dem Öffnen siehst du die **Startseite** mit allen aktiven Meldungen. Du kannst zwischen **Listenansicht** und **Kartenansicht** wechseln.

#### Listenansicht

Jede Meldung wird als Karte dargestellt mit:

- Foto (oder Platzhalter)
- Status-Badge (🔴 Vermisst / 🟢 Gefunden)
- Tierart-Badge
- Titel
- Favoriten-Herz

Klicke auf eine Karte, um die **Detailansicht** zu öffnen.

#### Kartenansicht

Alle Meldungen werden als Marker auf einer interaktiven Karte angezeigt. Klicke auf einen Marker, um die Kurzinfos zu sehen, und dann auf die Karte, um die Details zu öffnen.

### Filter

Über der Ergebnisliste stehen Filter zur Verfügung:

| Filter | Optionen |
|--------|----------|
| **Status** | Alle · Vermisst · Gefunden |
| **Tierart** | Hund · Katze · Vogel · … |
| **Rasse** | Wird dynamisch nach Tierart geladen |
| **Geschlecht** | Männlich · Weiblich · Unbekannt |
| **Farben** | Mehrfachauswahl per Checkboxen (aufklappbar) |
| **Ort + Umkreis** | Ortsname eingeben → Autocomplete → Radius wählen (5 / 10 / 25 / 50 km) |
| **Freitextsuche** | Durchsucht Titel und Beschreibung |

Alle Filter können über **„Filter zurücksetzen"** auf den Standard zurückgesetzt werden.

### Suche speichern

1. Gewünschte Filter setzen.
2. Auf **„Suche speichern"** klicken.
3. Einen Namen für die Suche vergeben (max. 100 Zeichen).
4. Die Suche erscheint im Menü unter **Gespeicherte Suche** (max. 20 pro Account).

## Meldung erstellen

!!! info "Anmeldung erforderlich"
    Zum Erstellen einer Meldung musst du angemeldet sein.

1. Menü → **Melden** (oder Tab „Melden").
2. Fülle das Formular aus:

### Foto (optional, empfohlen)

- Klicke auf das Upload-Feld und wähle ein Bild (JPG/PNG).
- Das Foto wird automatisch komprimiert.
- Optional: Klicke auf **„🤖 KI-Rassenerkennung starten"**, um Tierart und Rasse automatisch erkennen zu lassen.

### Pflichtfelder

| Feld | Beschreibung |
|------|-------------|
| **Name / Titel** | z. B. „Grauer Kater vermisst" (max. 50 Zeichen) |
| **Tierart** | Auswahl per Dropdown |

### Optionale Felder

| Feld | Beschreibung |
|------|-------------|
| **Rasse** | Wird nach Tierart-Auswahl geladen |
| **Geschlecht** | Männlich / Weiblich / Unbekannt |
| **Beschreibung** | Merkmale, Verhalten, Hinweise (max. 2000 Zeichen) |
| **Farben** | Mehrfachauswahl per Checkboxen |
| **Ort** | Ortsname eingeben → Mapbox-Autocomplete wählt Standort |
| **Datum** | Standardmäßig heute |

### KI-Rassenerkennung

1. Foto hochladen.
2. **„🤖 KI-Rassenerkennung starten"** anklicken.
3. Datenschutz-Hinweis bestätigen (Foto wird lokal analysiert, nicht extern gesendet).
4. Ergebnis prüfen: Tierart, Rasse und Konfidenzwert werden angezeigt.
5. **Übernehmen** füllt die Felder automatisch aus, oder **Ablehnen** für manuelle Eingabe.

### Speichern

Klicke auf **Speichern**. Die Meldung erscheint sofort in der Übersicht.

---

## Detailansicht einer Meldung

Klicke auf eine Meldungskarte, um die Details zu öffnen:

- **Foto** in voller Größe
- **Status**, **Tierart**, **Rasse**, **Geschlecht**, **Farben**
- **Beschreibung** und **Datum**
- **Standort**
- **Ersteller** (Benutzername + Profilbild)
- **Kontaktdaten** (falls vom Ersteller angegeben) – über den Kontakt-Button
- **Kommentare** (siehe nächster Abschnitt)
- **Favoriten-Button** (Herz)

---

## Kommentare

### Kommentare lesen

Kommentare sind für alle sichtbar – auch ohne Anmeldung.

### Kommentar schreiben (Anmeldung nötig)

1. In der Detailansicht das Textfeld unten nutzen.
2. Text eingeben (max. 1000 Zeichen, Shift+Enter für Zeilenumbruch).
3. Absenden.

### Antworten

Klicke **„Antwort"** unter einem Kommentar, um direkt darauf zu antworten. Antworten werden eingerückt unter dem Elternkommentar angezeigt.

### Reaktionen

Klicke auf ein Emoji unter einem Kommentar, um zu reagieren:

👍 ❤️ 😂 😮 😢

Erneut klicken entfernt die Reaktion. Die Anzahl der Reaktionen wird neben dem Emoji angezeigt.

### Kommentar löschen

Eigene Kommentare können über das Papierkorb-Symbol gelöscht werden (mit Bestätigung).

---

## Favoriten

### Hinzufügen

Klicke das **Herz-Symbol** (♡) auf einer Meldungskarte – das Herz wird rot gefüllt. Funktioniert in der Listenansicht, Kartenansicht und Detailansicht.

!!! info "Anmeldung erforderlich"
    Favoriten können nur von angemeldeten Nutzern gesetzt werden.

### Entfernen

Klicke erneut auf das gefüllte Herz – die Meldung wird aus den Favoriten entfernt.

### Favoritenliste anzeigen

Menü → **Favoritenlisten**. Hier siehst du alle gespeicherten Favoriten als Kartenansicht.

---

## Gespeicherte Suchen

### Suche anwenden

1. Menü → **Gespeicherte Suche**.
2. Klicke auf eine gespeicherte Suche.
3. Die Filter werden sofort in der Übersicht angewendet.

### Suche löschen

Klicke das **Löschen-Symbol** auf der gespeicherten Suche und bestätige den Dialog.

---

## Profil bearbeiten

Menü → **Profil bearbeiten**

### Benutzername ändern

1. Neuen Namen ins Textfeld eingeben.
2. **Speichern** klicken.
3. Der Name wird überall aktualisiert (Meldungen, Kommentare).

### Profilbild

- **Hochladen**: Klicke auf das Upload-Feld und wähle ein Bild.
- **Löschen**: Klicke **„Bild löschen"** — es wird der Standard-Avatar angezeigt.

Das Profilbild erscheint neben deinem Namen in Meldungen und Kommentaren.

---

## Meine Meldungen

Menü → **Meine Meldungen**

Hier werden alle selbst erstellten Meldungen angezeigt.

### Meldung bearbeiten

1. Klicke das **Bearbeiten-Symbol** (✏️) auf der Karte.
2. Ändere die gewünschten Felder (Titel, Beschreibung, Ort, Farben, Foto, …).
3. **Speichern**.

### Als gelöst markieren

Klicke **„Als gelöst markieren"** — die Meldung wird als **Wiedervereint** markiert:

- Status wechselt zu „Wiedervereint"
- Datum wird auf heute gesetzt
- Die Meldung verschwindet aus der aktiven Suche
- Bleibt in deinen eigenen Meldungen sichtbar

### Meldung löschen

Klicke das **Papierkorb-Symbol** und bestätige. Die Meldung wird unwiderruflich gelöscht (inkl. Fotos und Kommentare).

### PDF-Export

Klicke **„Exportieren als PDF"** auf einer eigenen Meldung. Das PDF enthält alle Details, das Foto und Kontaktdaten – ideal zum Ausdrucken oder Teilen.

---

## Einstellungen

Menü → **Einstellungen**

### Design (Hell-/Dunkelmodus)

Klicke das **Sonnen-/Mond-Symbol** in der AppBar, um zwischen Hell- und Dunkelmodus zu wechseln.

### Passwort ändern

1. Klicke **„Passwort ändern"**.
2. Aktuelles Passwort eingeben.
3. Neues Passwort eingeben und bestätigen (gleiche Regeln wie bei Registrierung).
4. **Speichern**.

### Konto löschen

1. Klicke **„Konto löschen"**.
2. Bestätige den Dialog.
3. Alle Daten werden gelöscht: Meldungen, Fotos, Kommentare, Favoriten, gespeicherte Suchen.

!!! warning "Nicht rückgängig machbar"
    Die Kontolöschung ist endgültig. Alle Daten werden sofort und unwiderruflich entfernt.

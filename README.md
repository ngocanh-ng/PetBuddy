# PetBuddy

PetBuddy ist eine Webanwendung für vermisste und gefundene Haustiere.
Nutzer können Meldungen erstellen, Fundtiere auf einer Karte entdecken und per KI die Rasse erkennen lassen.

**Tech-Stack:** Flet · FastAPI · Supabase · PyTorch
